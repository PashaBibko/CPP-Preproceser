LX Syntax

Comments

Comments in LX are marked with a '#' symbol. The comment either ends at the end of that line or when the next '#' symbol is found.

# This is a comment
# This is a comment # This is not a comment

Comments are currently ignored by the preprocessor. In the future they will be be inserted into the generated C++ code as comments.


Variable declaration

Variables in lx types are hard coded. This means that the type of the variable cannot be changed at runtime. This is because C++ is a statically typed language.
Also similiar to C++ you can define a variable without a value. This is useful for when you want to define a variable but you don't have a value for it yet. Note that the variable will be undefined until you assign a value to it.

int var1 = 0;
int var2; # Undefined


Types

Int and Float

Int and Float are both reprsentations of numbers. Int is a whole number and float is a decimal number.

int var1 = 0;
float var2 = 0.0;

# var3 will have a contents of 3 due to truncation
int var3 = 3.1459

Ints are defined by the 'int' keyword and floats are defined by the 'float' keyword. You can also make it unsigned by adding the 'unsigned' keyword before the type. This means that the number cannot be negative.

Bool

Bool is a representation of a true or false value. It is defined by the 'bool' keyword.

The values are 'true' and 'false'. The first letter can be capitalized or lowercase. You can also use '1' and '0' to represent true and false.

# All variables store true
bool var1 = True;
bool var2 = true;
bool var3 = 1;

Char and String

Char is a single ASCII character. If you do not know what ASCII is find out here.
It is defined by the 'char' keyword.

String is a collection of characters. It is defined by the 'string' keyword. You can access the individual characters of a string by using the [] operator.

# Stores the letter 'a'
char var1 = 'a';

string var2 = "Hello";
char var3 = var2[0];

Void

Void is a type that represents nothing. It is defined by the 'void' keyword. It is used for functions that do not return a value.

You can also use void as a type for a variable. This means that the variable does not have a type. This is useful for when you want to store a pointer to any type (however it is recommended to use a base class pointer).


If and Else Statements

If statements are used to run a certain scope of code if a condition is true. Else statements are used to run a certain scope of code if the condition is false. Elif statements are used to run a certain scope of code if the previous condition is false and the current condition is true.

The keywords use the corresponding 'if', 'else', and 'elif' keywords. Their conditions are not required to be in parentheses but must be followed by a colon. The scope of the code is defined by curly brackets unless the scope is only one line.

# One line If statement example
if var1 == 0:
  var1 = 1;

# You can still use brackets if you prefer
elif (var1 == 1):
{
  var1 = 2;
}

# Else statements cannot have any conditions
# That is the purpose of the elif statement
else:
{
  var1 = 0;
}

While Loops

While loops are used to run a certain scope of code while a condition is true. It follows the same syntax as the if statement.

while True:
{
  console::print("Who loves infinite loops?");
}

For Loops

For loops are used to run a certain scope of code a certain number of times. It follows the same syntax as the while loop for the contents of the loop.

By default the for loop starts at 0 and goes up to the number before the number you specify (iterating by 1 each time).

for int i count [5]:
{
  console::print(i, ", ");
}

# This will print 0, 1, 2, 3, 4,

You can also specify the starting number and the increment.

# Starts on 1
# Goes up to 5
# Iterates by 2

for int i count [1, 5, 2]:
{
  console::print(i, ", ");
}

Range based for loops

Instead of iterating through a range the size of the container you can iterate through the container itself. For each iteration you will be given a reference to the current element. This means any changes you make to the element will be reflected in the container.

int[] arr = [1, 2, 3, 4, 5];

for int i in arr:
{
  console::print(i, ", ");
}

Binary Logical Operators

Binary logical operators are used to compare two values. They return a boolean value.

There are two sets of binary logical operators. The first set is the comparison operators (!=, ==, <, >, <=, >=). The second set is the logical operators (and, or, not) which can also be written as (&&, ||, !).

Comparison operators are used to compare two values of the same type. == and != are the only operators that can be used on all types. The rest can only be used on arethmetic types.

Logical operators are used to compare two boolean values. 'and' returns true if both values are true. 'or' returns true if at least one value is true. 'not' returns true if the value is false.
